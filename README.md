# FastAcquisition2
 Fast acquisition plugin for Micromanager 2.0gamma
 
INSTALLATION

Copy FastAcquisition2.jar to micromanager/mmplugins/
Copy swing-layout-1.0.4.jar to micromanager/plugins/Micro-Manager/
